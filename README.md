# ADHDharma Three-Pot Tracker

Energy tracking tool for neurodivergent individuals.

## Features

- Track energy across Physical, Cognitive, and Emotional/Sensory domains
- User authentication with Supabase
- Activity tagging and pattern learning
- Historical data tracking
- Warning system for depletion zones

## Deployment

This app is configured for Vercel deployment.

1. Push to GitHub
2. Connect repository to Vercel
3. Deploy automatically

## Environment

No environment variables needed - Supabase credentials are in the code for now.
For production, move to environment variables.
